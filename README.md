## 🧠 Overview

This project analyzes emerging patterns in the global AI job landscape using real-world data from Kaggle. It focuses on:
- 💸 Salary distributions
- 🌍 Geographical spread
- 🧑‍💻 Experience levels
- 🏡 Rise of remote/hybrid work

---

| File | Description |
|------|-------------|

| `AI_Job_Market_Report_Arpita_Pani.docx` | Final report summarizing key findings and trends |

---

## 🔍 Highlights

- 🌎 **North America & Europe** lead in AI salaries
- 🏠 Over **60% of AI jobs are remote or hybrid**
- 🧑‍💼 **Executives earn up to 3x** more than entry-level roles
- 🐍 **Python, SQL, TensorFlow** are among the most in-demand skills
